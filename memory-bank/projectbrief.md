# Project Brief: FlashNotify

FlashNotify is an Academic Notification System built with Flask. It allows sending notifications to users (students/staff) with different priority levels and emergency types.

## Core Features
- Send notifications (Academic, Emergency, Info).
- Dashboard for viewing notifications and statistics.
- Admin panel for managing users and queues.
- Async task processing (threading/Celery).

## Goals
- Host the application online for access.
